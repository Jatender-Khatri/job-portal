# Job Portal
A web app to manage your Jobs posts efficiently. This is my second project using jsp and servlet.
## Features
- Admin and Normal both
- Create, Update, View & Delete Jobs
- Sign In and Sign Up
- Full Secure
## Client Side
- HTML
- CSS
- Boostrap
- JSP
## Server Side
- Java
- Servlet
## Database
- MySql
